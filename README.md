## 📂 Проекты

- **[Анализ поведения пользователей сервиса аренды самокатов](https://github.com/n3dsky/stats-sharing)**  
  Полный статистический анализ данных GoFast: EDA, расчёт бизнес-метрик, сегментация пользователей и проверка гипотез. Выявил различия между подписчиками и обычными пользователями, влияющие на выручку и удержание.

- **[Предсказание удоя и качества молока у коров](https://github.com/n3dsky/ml-linear-cows)**  
  Построил линейную регрессию для прогноза годового удоя и логистическую регрессию для оценки вкуса молока. Применил отбор признаков и метрики precision/recall для минимизации рисков в сельском хозяйстве.

- **[Выбор региона для бурения нефтяных скважин с оценкой рисков](https://github.com/n3dsky/ml-wells-research)**  
  Разработал линейную регрессию для предсказания объёмов нефти. С помощью Bootstrap оценил прибыль и риски убытков. Рекомендовал оптимальный регион с риском убытков < 2.5% и средней прибылью ~467 млн руб.

- **[HR-аналитика: предсказание удовлетворённости и оттока сотрудников](https://github.com/n3dsky/hr-analytics)**  
  Построил модели регрессии (уровень удовлетворённости) и классификации (отток) с использованием SVM. Выявил ключевые факторы оттока и дал рекомендации по удержанию персонала.

- **[Исследовательский анализ рынка недвижимости Санкт-Петербурга](https://github.com/n3dsky/spb-estate)**  
  Глубокий EDA архива объявлений Яндекс.Недвижимость: предобработка, создание признаков, выявление аномалий и факторов, влияющих на цену. Подготовил основу для системы оценки недвижимости и обнаружения мошенничества.

- **[Анализ российского кинопроката и эффективности государственной поддержки](https://github.com/n3dsky/cinema-review)**  
  Комплексный анализ кассовых сборов, окупаемости фильмов с господдержкой, жанровых и сезонных трендов. Сформулировал рекомендации по повышению эффективности господдержки кино.

- **[Предсказание медианной стоимости домов на PySpark и MLlib](https://github.com/n3dsky/big-data-ml)**  
  Реализовал полный пайплайн машинного обучения на Apache Spark: предобработка данных, масштабирование, обучение линейной регрессии с помощью MLlib и оценка качества (RMSE, R²).

- **[Персонализация предложений для клиентов интернет-магазина](https://github.com/n3dsky/practicum_git_project)**  
  Задача supervised learning: сегментация клиентов и предсказание покупательской активности. Лучшая модель — SVM. Проанализировал вклад признаков с помощью SHAP.

- **[Сервис предсказания риска сердечного приступа на FastAPI](https://github.com/n3dsky/heart-attack-prediction)**  
  Обучил модель CatBoost и развернул её как веб-сервис. Поддерживает одиночные, пакетные и CSV-предсказания с возвратом вероятности риска. Демонстрирует навыки от модели до production-ready API.
